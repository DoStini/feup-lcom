Shoots a ray from a given position in a given direction.

Uses the DDA algorithm (described in more detail here: https://lodev.org/cgtutor/raycasting.html) to traverse the map and check collision with walls. Can be used with a range.

Parameters
pospointer to the starting position of the ray
ray_dirpointer to the direction of the ray
ray_hitpointer to a ray_hit_t structure. The return of this function is stored here.
rangethe maximum range of the wall (in blocks), 0 if unlimited
